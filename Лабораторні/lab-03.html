Лабораторна робота № 3.
Використання методу табличного верстання

Мета роботи

Розглянути приклади табличного верстання та створити власну сторінку за таким зразком. Визначити недоліки табличного верстання.

Обладнання

Персональний комп'ютер. Текстовий редактор Sublime Text 3 або Notepad++. Web-браузер Chrome, Firefox, Opera, Internet Explorer*

Хід роботи

  1. Відкрити редактор коду та створити новий файл, якщо у вікні редактора вже завантажено будь-який інший

  2. Зберегти файл з іменем index.html

  3. Створити сторінку відповідно до макету

  4. У файл index.html помістити наступні рядки:

    <!DOCTYPE html>
<html>
  <head>
    <title>Табличная вёрстка</title>
  </head>
  <body>
    <table border="1" cellpadding="0" cellspacing="0" width="100%">
      <tr>
        <th colspan=2>шапка сайта (логотип, слоган, телефон)</th>
      </tr>
      <tr>
        <th width="20%">навигация</th>
        <th width="80%">заголовок</th>
      </tr>
      <tr>
        <td width="20%">
          <ul>
            <li><a href="index.html" title="Ссылка 1">Ссылка 1</a></li>
            <li><a href="index.html" title="Ссылка 2">Ссылка 2</a></li>
            <li><a href="index.html" title="Ссылка 3">Ссылка 3</a></li>
          </ul>
        </td>
        <td width="80%">контент</td>
      </tr>
      <tr>
        <td colspan=2>Низ сайта (баннеры, счетчики, информация)</td>
      </tr>
    </table>
  </body>
</html>

  5. Впевнитися, що результат відповідає наступному (7 - 9)

  6. Перевірити даний файл у валідаторі розмітки W3C за посиланням https://validator.w3.org/

  7. Виявити наявність помилок та попереджень. Скорегувати документ, використовуючи стилі CSS.

    Для розміщення стилів використати контейнер <style></style>

    Для налаштування ширини та висоти таблиці та її колонок використати стилі height i width

  8. Зберегти файл та перевірити його знову
  9. Створити HTML-сторінку за зразком сторінки Wikipedia на будь-яку обрану тему. Для створення каркасу сторінки використати таблицю, для налаштування зовнішнього вигляду - стилі CSS.
  10. Зберегти файл та перевірити його на валідність
  11. HTML-код файлу та скріншот перевірки валідатором останнього завдання зберегти у звіт
  12. Роздрукувати звіт та письмово відповісти на контрольні запитання
